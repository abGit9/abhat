Web Application
I started coding on the android platform. Implementing custom animations on my phone is a passion of mine. Web development has provided another space to do this. Simply put, the web is a larger canvas upon which I can make my animations bigger and bolder. A goal of my websites, like my android applications, is to make the visual experience as seamless and smooth as possible.
Anime.js
Anime.js is a javascript library I used to streamline the implementation of animations throughout this web application. It provides an abstraction that makes it easy to change multiple properties of object(s) simultaneously in synchrony.
Custom Anims
The landscape was implemented without using anime.js. A timer was used to periodically update object properties and refresh the frame. If you move your pointer over a bird or a tree, relevant attributes and values are displayed in real time.
Languages
Teaching myself java and xml for android made learning web development easier. More specifically, in android, java is used to manipulate user interfaces that are structured by xml. This is similar to web development where web pages are organized by html but controlled by javascript. In both cases, markup languages are parsed so that programming languages can bring movement and action to the user interface.
